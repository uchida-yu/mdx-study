## Component
### コンポーネントが渡せる
```jsx
<Example components={
    {
        Planet: () => <span style={{ backgroundColor: 'aquamarine' }}>component</span>
    }
} />
```
<Planet />  

### マークダウンの要素を書き換えられる
- em  
Rewrite `em`s (`*like so*`) to `i` with a goldenrod foreground color => *like so*  
- link  
[他の要素はこちらで確認](https://mdxjs.com/table-of-components/ "title")  

### テーマを使う

## MDXProvider
MDXProvider を使用するとcomponentsをバケツリレーしなくて済む
```jsx
import { MDXProvider } from '@mdx-js/react'

<MDXProvider components={{h1: Component1, h2: Component2}}>
    <Content />
</MDXProvider>
```
